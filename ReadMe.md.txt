after compiling all files
run one instance of ServerGraphic.java (not Server.java) for having GUI for your server, if you'd
prefer a simple server, Run Server.js. then start the server (you can change the port number as you wish)

after running a server, run as many instances as you want of ClientGraphic.java (not Client.java)for having GUI for your client, if you'd prefer a simple client, Run Client.js
master branch of this project has no user authentication for ease of use.
and error handling is done in the simplest way :) .
you can clone authentication branch for authentication.
